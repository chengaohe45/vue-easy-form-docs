    # 项标签/label

    字段：label
    值类型有：

    • false： 默认值，label隐藏，空间不存在的
    • ''： 空值 但label空间是存在的
    • string： 项label文字；支持动态解析
    • object： 一个对象，见组件格式

    # 组件详解

    属性名 说明 类型 可选值 默认值
    hidden 控制组件是否隐藏, 支持动态解析 boolean -- false
    align 文本的方向 string left、center、right --
    flex label的长度控制 string ""： label的长度将会是labelWidth

    full： 项中有多少点多少。此值一般用于component

    self： label的文本占多宽就多宽。此值一般用于label    		 ""
    其它 跟组件格式一样 -- -- --
