    # tabs布局

    字段（见行高亮）：



     
     
     
     
     

     
     









    propName: {
  label: "label值", 
  layout: {
    name: "tabs",  // 值："tabs","space"; 当是"space",说明是一个占位空间,其它值设置是无效的
    type: "card",  // 类型："bg","card","line"; 默认为card
    hasBorder: true,
    padding: undefined
  },
  title: "标题",
  properties: {
    name: {
      label: "名称",
      component: "el-input",
      value: "天天"
    }
    // ... 其它项
  }
}

    # 实例

    tabs布局

    1. tabs的头部显示的是label的文本，并不是title,当label为空或不设置时，则会显示该项的key；
    2. 占位空间若作为tabs的一项，将会忽略；
    3. tabs的项存在分组，将会忽略。
