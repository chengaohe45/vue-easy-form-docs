# 1.7.9

  • 优化分组布局.

# 1.7.8

  • 增加：组件事件和组件验证返回表单实例，这样在箭头函数中使用表单实例就更加方便和灵活；
  • 修复：若在actions中直接配置事件，导致无法触发在元素中直接配置的事件的问题.

# 1.7.7

  • 优化数组：增加delWarnBtns，控制删除警告框按钮的文本

# 1.7.6

  • 优化数组：增加btnType，控制操作按钮的显示方式
  • 优化数组：数组下方的全删和增加按钮统一改为图标形式

# 1.7.5

  • 优化表单方法：重置方法(reset)增加重置类型参数

# 1.7.4

  • 优化组件格式：增加组件插槽

# 1.7.3

  • 增加数组删除提示框可自定义：增加delMsg, delAllMsg
  • 增加数组操作钩子：增加before钩子
  • 优化块组件验证：修复required无动态解析问题
  • 优化组件数组头部显示：修复subLabel无动态解析问题

# 1.7.1

  • 优化动态解析：优化数组es写法

# 1.7.0

  • 优化动态解析：es写法可以去掉大括号
